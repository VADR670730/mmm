Website Event Role

  • The purpose of this module is to add the event roles in Events.

Prerequisites before Installation

  • No prerequisites are required before the installation.

Configurations

  • Events -> Configuration -> Settings -> Event Role Management.

  • Select Manage Role Events for managing the roles.

Step-By-Step Guidelines

After the cofiguration now we create event roles from Events -> Configuration -> Events Roles

In Events, if Role Selection Mandatory(Boolean) is selected then the events roles field will be mandatory in Events.

In Event Registration form, we will manage one event roles field and that is filter according to the event we selected.
