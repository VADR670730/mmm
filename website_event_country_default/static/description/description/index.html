Event Default Country in Website

  • This module will set the default country of the event in the address form of website when any customer wants to attend any event.

Prerequisites before Installation

  • No prerequisites are required before the installation.

Configurations

  • No Configurations are required.

Step-By-Step Guidelines

Let suppose, we created one event i.e Technical Training and we set the location of that event as Luxembourg which will be shown in below screen.

Now when the customer wants to book their seat in Technical Training event then the default country of the event had been selected as Luxembourg in the address form of the customer page.
