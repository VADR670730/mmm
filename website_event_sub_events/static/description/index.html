Website Event Sub Events

  • The purpose of this module is to add the sub events in Events with respect to event roles.

Prerequisites before Installation

  • No prerequisites are required before the installation.

Configurations

  • No Configuration Required.

Step-By-Step Guidelines

You can create the record of sub events in Sub Event tab of Event and also select the event roles with respect to that Event. You can also assigned the price of that particular sub events.

One Smart Button of Sub Events is also manage where it will display the numbers of sub events in particular event and also we create the sub events record from that smart button.

In the event registration form you can select sub events on the basis of events and Total price will be changed on the basis of Event ticket and Sub Event.

In online event registration form, you can select sub events on the basis of events and Total price will be changed on the basis of Event ticket and Sub Event.
